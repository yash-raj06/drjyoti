# Dr Jyoti Next.js V3

This is the polished V3 version of the clinic website starter.

## Added in V3
- `next/image` support for hero and gallery images
- Sticky mobile CTA bar
- Google Maps embed placeholder
- Formspree-ready contact form
- Better homepage visual structure

## Start
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run start
```

## Replace before launch
- `public/clinic-hero.jpg`
- `public/gallery-1.jpg`
- `public/gallery-2.jpg`
- `public/gallery-3.jpg`
- phone, WhatsApp, form endpoint, and map URL in `lib/site-data.ts`
- domain references in metadata and sitemap

## Deploy
1. Push to GitHub.
2. Import into Vercel.
3. Deploy.
4. Add real domain and update metadata.
